## 🔗 Linked List
각 노드가 데이터와 포인터를 가지고 한 줄로 연결되어 있는 방식으로 데이터를 저장하는 자료구조.

## 🕶 Features
1. 데이터의 추가/삽입 및 삭제가 용이함
2. 연속된 메모리 공간 할당이 쉬움
3. 인덱스가 없이 그저 연결관계만 있기 때문에 특정 노드를 불러내기 어려움
